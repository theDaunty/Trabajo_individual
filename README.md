## 📋 Descripción del Proyecto

Aplicación web que implementa tres algoritmos de procesamiento de imágenes trabajando directamente a nivel de píxel:

| Inciso | Descripción |
|--------|-------------|
| **A** | Clasificación de texturas (césped, tierra, cemento, asfalto) |
| **B** | Filtro de suavizado promedio 3×3 |
| **C** | Cover "La Vaca Lola" con animación sincronizada (estilo Bad Bunny) |

## 🚀 Instalación y Ejecución

### Requisitos previos

- Python 3.13 o superior
- pip (gestor de paquetes de Python)

### Pasos para ejecutar

```bash
# 1. Clonar el repositorio
git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
cd TU_REPOSITORIO

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar la aplicación
python app.py
